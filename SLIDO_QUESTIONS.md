# Slido Questions for AI-Native Productivity & Automation Bootcamp

Use these questions to make the 5-day training more interactive. They are grouped by day and moment so you can import or copy them into Slido as needed.

## Suggested Slido Setup

- Create one Slido event per day, or one event with sections named Day 1 to Day 5.
- Enable anonymous responses for questions that ask about confidence, concerns, or blockers.
- Keep Q&A open all day with moderation enabled.
- Use word clouds for warmups, polls for quick checks, ranking for prioritization, and open text for reflection.

**Type:** Ranking

**Question:** Which AI risk worries you most in public sector or workplace use?

### Responsible AI & Model Awareness

**Type:** Multiple choice

**Question:** Before using an AI-generated answer in an official document, what should happen first?

**Options:**

- Verify the answer against a trusted source.
- Copy it if it sounds professional.
- Ask the model to make it more confident.
- Remove all citations to make it cleaner.
